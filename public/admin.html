◉ Configurar Chat com Gemini AI
O chat do site usa o Gemini 2.0 Flash do Google. Para ativá-lo você precisa de uma API Key gratuita.

→ Obter API Key no Google AI Studio (gratuito)
1
Acesse aistudio.google.com e faça login com sua conta Google.
2
Clique em "Get API Key" → "Create API key". Copie a chave.
3
Abra o arquivo server.js e localize:
const GEMINI_API_KEY = process.env.GEMINI_API_KEY || 'SUA_API_KEY_AQUI';
Substitua SUA_API_KEY_AQUI pela sua chave, ou use a variável de ambiente GEMINI_API_KEY.
4
Salve e reinicie: node server.js. Chat ativado!
